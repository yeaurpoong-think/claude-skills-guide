💡 WHAT IS CLAUDE SKILLS

스킬이 뭔가요?

Anthropic이 Claude Code에 공식 도입한 확장 시스템. .md 파일로 만든 커스텀 명령을 Claude에게 가르쳐서, 특정 분야의 전문가처럼 작동하게 만듭니다.

📦
AI를 위한 npm
스킬 = AI 에이전트의 패키지. npm처럼 원하는 능력을 골라서 설치하고, 필요 없으면 제거합니다.
🔗
오픈 스탠다드
Claude Code뿐 아니라 Cursor, Codex, Windsurf, Gemini CLI 등 주요 AI 에이전트 전체에서 동작합니다.
📁
파일 기반 관리
~/.claude/skills/ 폴더에 넣으면 전체 적용, .claude/skills/에 넣으면 해당 프로젝트만 적용.
⚡
슬래시 커맨드로 호출
설치 후 /skill-name 또는 @skill-name 으로 즉시 호출. 배우는 시간 필요 없이 바로 씁니다.
🚀 INSTALLATION GUIDE

설치 방법,
3가지 중 하나면 됩니다

Claude Code가 설치된 환경에서 진행하세요. 가장 빠른 건 방법 A — 명령어 한 줄이면 끝납니다.

1
터미널에서 명령어 실행
Claude Code CLI가 설치된 터미널을 열고 아래 명령어를 붙여넣으세요. plugin install 명령 하나로 다운로드→설치가 자동으로 됩니다.
# 예시: Marketing Skills 설치
$ claude-code plugin install https://github.com/coreyhaines31/marketingskills

# Superpowers 설치
$ claude-code plugin install https://github.com/obra/superpowers-lab

# Remotion Skills 설치
$ claude-code plugin install https://github.com/remotion-dev/skills
2
설치 확인
설치가 완료되면 Claude Code를 재시작하거나 새 대화를 시작하세요. / 를 입력하면 설치된 스킬 목록이 자동완성으로 표시됩니다.
✓ marketingskills installed
✓ Available as /marketing, /copywriting, /seo ...
1
GitHub에서 SKILL.md 파일 다운로드
원하는 스킬의 GitHub 저장소로 이동해서 SKILL.md 또는 skills/ 폴더의 파일을 다운로드하세요.
2
스킬 폴더에 파일 배치
다운로드한 파일을 아래 경로에 넣으면 됩니다. 폴더가 없으면 직접 만드세요.
# 전체 적용 (개인 설정)
~/.claude/skills/스킬이름/SKILL.md

# 현재 프로젝트만 적용
.claude/skills/스킬이름/SKILL.md
3
Claude Code 재시작 후 사용
/스킬이름 으로 바로 호출하세요. SKILL.md 안에 정의된 모든 커맨드가 활성화됩니다.
1
저장소 클론
스킬 컬렉션 전체를 한 번에 받아서 원하는 것만 골라 쓰는 방법입니다. 345개 스킬이 들어있는 컬렉션도 있어요.
$ git clone https://github.com/alirezarezvani/claude-skills
# ⭐ 22,700 · 345개 스킬 모음집
2
원하는 스킬만 복사
클론된 폴더에서 원하는 스킬 디렉토리를 ~/.claude/skills/ 로 복사하면 됩니다.
$ cp -r claude-skills/marketing ~/.claude/skills/marketing
$ cp -r claude-skills/remotion ~/.claude/skills/remotion
✓ Done! 2 skills installed

❓ FAQ

자주 묻는 것들

Claude Skills는 현재 Claude Code(터미널 기반 AI 코딩 에이전트)에서 동작하는 기능입니다. claude.ai 웹 브라우저 버전에서는 직접 지원되지 않습니다. 단, Cursor나 Windsurf 같은 에디터에서도 같은 스킬 파일을 사용할 수 있어요.
skills.sh (Vercel 제작)가 사실상 공식 스킬 디렉토리 역할을 합니다. 905,000개 이상의 스킬이 등록되어 있고, 설치 수·인기 순위로 검색할 수 있어요. claudeskills.com은 큐레이션 마켓플레이스로 waitlist 운영 중입니다.
기본적으로 스킬은 독립적으로 동작해 충돌이 거의 없습니다. 다만 같은 커맨드 이름이 중복되면 마지막에 설치된 스킬이 우선합니다. 너무 많이 설치하면 컨텍스트 로딩 시간이 늘어날 수 있으니 자주 쓰는 것 위주로 10개 이내로 관리하는 게 좋아요.
당연히 가능합니다. SKILL.md 파일에 YAML 프론트매터와 마크다운 지시사항을 작성하면 됩니다. Agent Skills Open Standard를 따르면 Claude뿐 아니라 Cursor, Codex, Gemini CLI에서도 동작하는 범용 스킬을 만들 수 있어요.
현재 GitHub에 올라온 스킬들은 대부분 무료 오픈소스입니다. claudeskills.com 같은 마켓플레이스가 활성화되면 프리미엄 스킬도 등장할 수 있지만, 지금 단계에서는 사실상 전부 무료입니다.
